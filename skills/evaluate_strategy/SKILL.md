# evaluate_strategy

## Purpose

Evaluate strategy results, risk, robustness, and research validity.

## Expected Inputs

- Experiment context
- Backtest result
- Metrics and artifacts
- Benchmark definition

## Expected Outputs

- Evaluation summary
- Risk diagnostics
- Robustness checklist
- Recommendation and caveats

## Workflow

1. Check that the run config is reproducible.
2. Review performance, drawdown, turnover, exposure, and capacity assumptions.
3. Identify data leakage, overfitting, or fragile periods.
4. Summarize open questions before any recommendation.
